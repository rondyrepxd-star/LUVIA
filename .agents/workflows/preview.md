To get the preview link and ensure the server is running, follow these steps:

1. Check if the development server is already running on port 9002.
// turbo
2. Run `cmd /c npx next dev -p 9002` in the background if it's not running.
3. Provide the user with the following links:
   - Local: [http://localhost:9002](http://localhost:9002)
   - Network: [http://192.168.100.2:9002](http://192.168.100.2:9002)
   - *Note*: If you are using a cloud environment like IDX, please use the "Web Preview" tab in your IDE.
